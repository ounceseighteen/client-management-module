CRM Система "Азимут Плюс" - Модуль учета клиентов
Веб-приложение для учета клиентов, разработанное в рамках производственной практики в компании "Азимут Плюс". Система предоставляет REST API для работы с данными клиентов и современный веб-интерфейс.

Основные возможности
Добавление клиентов через веб-форму

Просмотр списка клиентов с пагинацией

Удаление клиентов одним кликом

Экспорт данных в различные форматы:

HTML (готовый для печати)

CSV (для Excel и аналитики)

REST API для интеграции с другими системами

Адаптивный дизайн для мобильных устройств

Технологический стек
Backend: Python 3.x, Flask, SQLAlchemy

Frontend: HTML5, CSS3, JavaScript (ES6)

База данных: SQLite

Стилизация: CSS Grid, Flexbox, градиенты

Иконки: Font Awesome 6

Установка и запуск
Клонируйте репозиторий или скопируйте файлы проекта

Установите зависимости:

bash
pip install -r requirements.txt
Запустите приложение:

bash
python app.py
Откройте в браузере:

text
http://localhost:5000
REST API Endpoints
GET /api/clients - получить список всех клиентов

POST /api/clients - добавить нового клиента

DELETE /api/clients/<id> - удалить клиента по ID

Пример добавления клиента через API:
json
{
  "name": "ООО Ромашка",
  "email": "info@romashka.ru",
  "phone": "+79991234567",
  "company": "Ромашка"
}
