# Модуль учета клиентов CRM-системы "Азимут Плюс"

Веб-приложение для управления клиентской базой с экспортом данных. Разработано в рамках производственной практики.

## Функции
- Добавление, просмотр, удаление клиентов
- Экспорт данных в PDF и CSV форматах
- REST API для интеграции
- Адаптивный веб-интерфейс

## Технологии
- Python 3.8+, Flask
- SQLite база данных
- HTML/CSS/JavaScript

# Установка и запуск

## Установка зависимостей

pip install -r requirements.txt

## Запуск приложения

python app.py
Приложение доступно по адресу: http://localhost:5000

##Структура проекта

app.py - основное приложение

database.py - работа с базой данных

templates/ - HTML страницы

static/ - CSS стили
